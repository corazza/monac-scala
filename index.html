Mona

functional programming with a focus on speed, safety, and control

View on GitHub Download .zip Download .tar.gz

Mona is a functional programming language with a focus on speed, safety, and control. The language is still in very early development.

I'm blogging about the language here: jancorazza.com/tag/mona/.

Page under construction.

Features

  • Pattern matching
  • Type inference
  • Algebraic types
  • Type classes
  • Haskell-inspired syntax
  • Low-level memory access
  • Unique reference semantics
  • Ownership and borrowing
